Cầu hình kết nối BHTĐ

- Chức năng : Cấu hình thông tin kết nối giữa chương trình điều khiển trung tâm S3M và chương trình quản lý lệnh theo vòi bơm S3-Scadar

- Đối tượng sử dụng : Admin

- Đường dẫn : Cấu hình CHXD/ Cấu hình kết nối BHTĐ

     + Host IP : địa chỉ IP cài đặt chương trình S3M
     + Host Port : cổng kết nối chương trình S3M
     + Access ID : 0 (tương ứng với Client ID khi kết nối với S3M, Scadar luôn là 0)
     + Access Token : Mặc định EAAFrIE64ZBMIBAP74JwV4Cbfbj9RoSP

------------------------------------------------------------------------------

Cấu hình vị trí vòi bơm (S3Config)

- Chức năng : Cấu hình vị trí vòi bơm hiên thị lên màn hình S3-Scadar. Những vòi bơm nào được khai báo trong phần này sẽ là những vòi bơm được điều khiển BHTĐ.

- Đối tượng sử dụng : Admin

- Đường dẫn : Cấu hình CHXD/ Cấu hình vị trí vòi bơm

------------------------------------------------------------------------------

Cầu hình thông tin OPT

- Chức năng : Cấu hình thông tin OPT kết nối với hệ thống. Mỗi OPT có thể kết nối với một nhóm vòi bơm và một vòi bơm có thể được điều khiển bởi nhiều OPT.

- Đối tượng sử dụng : Admin

- Đường dẫn : Cấu hình CHXD/ Cấu hình thông tin OPT

------------------------------------------------------------------------------

Cầu hình thông tin Ví điện tử

- Chức năng : Cấu hình thông tin kết nối thanh toán ví điện tử. Mỗi cửa hàng có 1 kết nối tới 1 loại ví điện tử tương ứng.

- Đối tượng sử dụng : Admin

- Đường dẫn : Cấu hình CHXD/ Cấu hình thông tin Ví điện tử
